people.apache.org has been decommissioned

The committer-index.html page has been superseded by the Apache Phonebook
Usage notes are available in the Apache Phonebook Description

For example you can look up the list of current committers

Or you can look up a committer directly as follows:
http://home.apache.org/phonebook.html?uid=user-id
This corresponds to the original URL:
http://people.apache.org/committer-index.html#user-id

There is also a list of Persons with signed CLAs but who are not (yet) committers. This is useful for checking if a CLA has been received and recorded in the iclas.txt file which is maintained by the secretary.

PGP keys of committers are available.
