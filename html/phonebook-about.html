Apache Phone Book application

Usage

As well as using the search fields to find projects or people, the application supports direct query access:

  • http://home.apache.org/phonebook.html?project=tom - matches all project names containing the string 'tom' (excluding the Apache prefix)
  • http://home.apache.org/phonebook.html?user=tom - matches all user names containing the string 'tom'
  • http://home.apache.org/phonebook.html?pmc=tomcat - find exact match for project (case-insensitive)
  • http://home.apache.org/phonebook.html?uid=tomcat - find exact match for user id (case-insensitive)

About

This site is generated from data sources on Whimsy:

  • PMC committee information
  • ASF member information
  • ICLA information
  • LDAP people information
  • LDAP Unix groups
  • LDAP committee groups
