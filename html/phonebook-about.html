Apache Phone Book application

About

This application shows the membership of PMCs and various LDAP groups used for granting karma. It can also show the groups to which a person belongs.

Note: for the purpose of this application, ASF members are considered to be a PMC, i.e. ?pmc=member shows the list of current ASF members. There is no LDAP committee group for members. There is an LDAP unix group (which is used to grant member karma)

Usage

As well as using the search fields to find projects or people, the application supports direct query access in the form:

http://home.apache.org/phonebook.html?type=name

The possible types are listed below; click on a link to run the sample query (opens in new tab)

  • ?project=tom - matches all project names containing the string 'tom' (excluding the Apache prefix)
  • ?user=tom - matches all user names containing the string 'tom'
  • ?pmc=tomcat - find exact match for project
  • ?pmc=member - show list of current ASF members
  • ?uid=fielding - find exact match for user id
  • ?unix=tomcat - find exact match for LDAP unix group
  • ?unix=committers - Show list of all committers
  • ?ctte=tomcat - find exact match for LDAP committee group
  • ?service=pmc-chairs - find exact match for LDAP service group
  • ?podling=hcatalog - find exact match for podling (note: not all podlings have groups)
  • ?other=xmlgraphics-fop - find exact match for non-LDAP group

Notes

Committers may provide homepage URLs in LDAP:
Login to https://id.apache.org/ and populate the "Homepage URL:" field.
Remember to provide your password (near the bottom of the page) before pressing submit

PMC membership is defined by the contents of the committee-info.txt file. The corresponding data file is committee-info.json.

All PMC members should be members of the corresponding LDAP committee group (see public_ldap_committees.json).
Membership of this LDAP group is used to grant karma for some parts of SVN that are private to the PMC and to give access to the PMC mail archives.
The LDAP committee group should not normally contain people who are not on the PMC.

All PMC members should also be members of the LDAP committer(unix) group. This group is used to grant karma for SVN.

Data Sources

This site is generated from the following data sources on Whimsy:

  • PMC committee information (committee-info.json)
  • ASF member information (member-info.json)
  • ICLA information - names and ids (icla-info.json)
  • LDAP people information - names and ids (public_ldap_people.json)
  • LDAP Unix groups (public_ldap_groups.json)
  • LDAP committee groups (public_ldap_committees.json)
  • LDAP Service groups - e.g. pmc-chairs (public_ldap_services.json)
  • non-LDAP groups - e.g. podlings) (public_nonldap_groups.json)
